# aco_practice_churn_prediction
Built an end-to-end ACO churn prediction system using synthetic data (500 practices, 150K patients). Includes 6-table model, EDA, stats tests, survival analysis, ML (RF/GBM/LR), SHAP, and segmentation. Achieved 78% ROC-AUC; key drivers: benchmark gap, downside risk, and practice size.
